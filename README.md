# 🚀 AI Carousel Generator

🔗 Live Demo: https://ai-carousel-generator-three.vercel.app/

## 📌 Overview
AI Carousel Generator is a full-stack web application that converts a simple idea into structured, editable, and downloadable social media carousel slides.

## 🎯 Features
- 🧠 AI-powered slide generation
- 🔄 Regenerate slides instantly
- ✏️ Edit slides in real-time  
- 📥 Download individual slides as images  
- 📋 Copy slide content  
- 🎨 Clean and responsive UI  

## 🛠️ Tech Stack
- Frontend: React (Vite)  
- Backend: Node.js, Express  
- API: AI integration  
- Libraries: Axios, html2canvas  

## ⚙️ How It Works
1. User enters an idea  
2. Backend generates structured slide content using AI  
3. Frontend displays slides  
4. User can edit and download slides  

## 🌐 Deployment
- Frontend: Vercel  
- Backend: Render
